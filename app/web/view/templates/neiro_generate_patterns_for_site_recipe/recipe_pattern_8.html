Помощь и инструкции

Основные функции

1. Добавление плиток: используйте панель "Плитки" для добавления новых элементов.

2. Перемещение: перетаскивайте плитки мышью для изменения их положения.

3. Редактирование: щелкните правой кнопкой мыши по плитке для вызова контекстного меню.

4. Соединения: создавайте связи между плитками через контекстное меню.

Панели управления

Цвет: настройка цветовой схемы рецепта.

Шрифт: выбор шрифтов и их размеров.

Плитки: добавление и настройка плиток.

Фон: изменение фонового изображения или паттерна.

Углы: настройка скругления углов плиток.

Зум: управление масштабом просмотра.

Расположение: автоматическое расположение плиток.

Интерфейс: настройка внешнего вида панелей управления.

Горячие клавиши

Ctrl + колесо мыши: изменение масштаба.

Shift + перетаскивание: выделение нескольких плиток.

Delete: удаление выбранных плиток.

Ctrl + C / Ctrl + V: копирование и вставка плиток.

Сохранение

Используйте кнопки в нижнем левом углу для сохранения рецепта в формате HTML или PNG.

Для копирования в буфер обмена используйте сочетание клавиш Ctrl+C после выделения.
